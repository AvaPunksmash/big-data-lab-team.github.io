We are looking for motivated, creative students with strong programming skills, a drive for research, and good reading & writing skills. If you are interested, please send your CV and a brief research statement explaining your research interests, or referring to one or more of the following themes to tristan.glatard@concordia.ca. Candidates deemed excellent will be interviewed. Reading the getting started guide is a good way to prepare for the interview. Generic emails will not be answered.

Big neuroinformatics pipelines

We design and evaluate architectures, frameworks and tools to process Big neuroimaging Data on high-performance computing platforms.

  • Software projects: Spark for Neuroimaging (sim), Clowdr.
  • Technologies: Apache Spark, Compute Canada, Amazon Web Services.
  • Publications: sim abstract.

Computational reproducibility

We design tools to evaluate the computational reproducibility and numerical stability of data processing pipelines.

  • Software projects: repro-tools.
  • Technologies: ReproZip, ltrace.
  • Publications: FINF paper, INCF abstract.

Web technologies for interoperable platforms

We develop technologies to enable global sharing of data and pipelines across neuroinformatics platforms.

  • Software projects: Boutiques, CARMIN.
  • Technologies: OpenAPI, Swagger, Docker, Singularity, JSON schema.
  • Publications: Boutiques paper, CARMIN abstract.

Ultra-high resolution imaging

We develop technical frameworks and algorithms to speed-up the processing of ultra-high resolution 3D images.

  • Software projects: sam.
  • Technologies: Hadoop Distributed File System, Nibabel.
  • Publications: IEEE BigData 2017.

Simulation of complex networks

We develop simulators of processes involved in complex networks such as computing networks, brain networks and supply-chain networks.

  • Software projects: soon coming.
  • Technologies: SimGrid.
  • Publications: CCGrid 2017, CCGrid 2013.

Connected objects and IoT

We develop data analytics and system architectures for energy-efficient and privacy-preserving data collection in the Internet of Things.

  • Software projects: soon coming.
  • Technologies: to be defined.
